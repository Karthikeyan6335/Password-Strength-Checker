# Password Strength Checker (Python)

# Overview
The Password Strength Checker is a simple Python-based tool that evaluates the strength of a password based on its length, variety of characters, and complexity.  
It helps users create stronger passwords and promotes secure authentication practices.

# Objective
To develop a Python program that analyzes user-input passwords and provides instant feedback on their strength, ensuring better password security awareness.

# Features
- Evaluates password strength based on predefined rules.  
- Checks for uppercase, lowercase, digits, and special characters.  
- Classifies passwords as Weak, Moderate, or Strong.  
- Simple and efficient command-line interface.

# Technologies Used
- **Language:** Python  
- **Library:** re (Regular Expressions)
